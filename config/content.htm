怎麼興建網站

(1)下載fall2020_cp.7壓縮檔   解壓縮到我的隨身系統(USB)

(2)下載 Portable GIT. 解壓縮並放入DATA中的PortableGit資料夾中

(3) 修改Start的路進

(4)到github 新建一個repository(資料庫)  名子為cad2020 

(5)到github網站git clone自己倉儲的網址clone完後git submodule add

(6)進入cmsimde  把up_dir裡的東西全部複製

(7)放到cmsimde外面  cad2020的目錄下

  (8)進入cmsimde目錄下   將wsgi.py拖入SciTE     tools-->go

(9)如果成功了 記得  git push 才能把網頁推上去 不然會顯示不出來!!!
